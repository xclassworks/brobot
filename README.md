### Como rodar ###

Configure os pinos do motor (velocidade, direção horária e direção anti-horária)

Faça upload no Arduino

## Mapa de Motores

### MOTOR A (Motor esquerda)
- Verde     -   Ativa MA
- Amarelo   -   IN1
- Vermelho  -   IN2

### MOTOR B (Motor direita)
- Verde     -   Ativa MB
- Amarelo   -   IN3
- Vermelho  -   IN4
